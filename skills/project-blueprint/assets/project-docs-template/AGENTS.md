# {{PROJECT_NAME}} AI 工作规则

## 每次任务开始

1. 读取 `docs/INDEX.md`。
2. 读取 `docs/PROJECT_STATUS.md`。
3. 读取当前 Phase 文档。
4. 根据索引路由仅读取任务相关规范。
5. 遵守仓库内更具体目录的指令文件。

## 每次任务结束

1. 更新项目状态、完成项、验证、阻塞和下一步。
2. 向当前阶段日志追加记录，不覆盖历史。
3. 公开行为变化时更新规范；重要决策新增 ADR。
4. 只报告实际执行的测试结果。

## Phase 结束

- 执行 Phase 验收命令。
- 写阶段结束报告并更新项目总日志。
- 更新项目状态并切换下一阶段。
